Ekstensi Domain

Pengertian Eksistensi Domain :

Ekstensi domain adalah bagian akhiran dari nama domain (misalnya, .com, .org, .id) yang mengidentifikasi jenis situs web atau negara asal. Jenis-jenis utamanya meliputi Generic Top-Level Domain (gTLD) seperti .com untuk komersial, Country Code Top-Level Domain (ccTLD) seperti .id untuk Indonesia, dan New Generic Top-Level Domain (ngTLD) yang lebih spesifik seperti .tech atau .online.

Jenis-Jenis Eksistensi Domain :

(1). Generic Top-Level Domain (gTLD) Penjelasan : Ekstensi yang umum digunakan untuk berbagai jenis organisasi.

Contoh-contoh gTLD :

(a). .com: Digunakan untuk tujuan komersial.

(b). .org: Digunakan untuk organisasi non-profit atau nirlaba.

(c). .net: Umumnya digunakan untuk organisasi yang berkaitan dengan jaringan internet.

(d). .edu: Dikhususkan untuk lembaga pendidikan.

(2). Country Code Top-Level Domain (ccTLD) Penjelasan : Terdiri dari dua huruf yang mewakili negara tertentu.

Contoh-contoh ccTLD :

(a). .id: Untuk Indonesia.

(b). .jp: Untuk Jepang.

(c). .us: Untuk Amerika Serikat.

(3). New Generic Top-Level Domain (ngTLD) Penjelasan : Ekstensi yang lebih baru dan lebih spesifik untuk niche tertentu.

Contoh-contoh ngTLD :

(a). .tech: Untuk situs web berfokus pada teknologi.

(b). .store: Untuk toko online.

(c). .blog: Untuk blog.
